Unofficial Mashable like Blippr plugin that lets users review anything directly from your blog.

== Description ==

This plugin allows your users to use Blippr reviews directly from your blog like Mashable does. 
Simply add the tag [blippr] to your posts to add an hot word that will be used to generate Blippr tooltip.

This plugins makes heavy use of Blippr APIs and Js and all credits go directly to their great developers.

This plugin is not an official release from Blippr staff, just a contribute from a fan.

== Installation ==

1. Upload `blippr.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use it from HTML editor or directly putting the [blippr] tag inside your posts

eg: [blippr]Wordpress[/blippr]

== Frequently Asked Questions ==

= How do I use this plugin? =

After plugin activation you shold see a new [blippr] button inside your html editor (while creating or modifying a page/post).
Use it to 'blippr' a word.
